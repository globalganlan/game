## 概述

全球感染（GlobalGanLan）的世界觀設定、故事線規劃、角色背景故事。目前遊戲尚無劇情系統，本 spec 為未來內容擴展規劃。

- `specs/hero-schema.md` — 英雄基礎資料（角色名稱/屬性/職業）
- `specs/stage-system.md` — 主線關卡章節架構
- `specs/ui-flow.md` — 劇情播放 UI 介面

## 一、世界觀

### 背景設定

- **時間**：近未來
- **事件**：不明病毒全球爆發，感染者異變為喪屍
- **玩家角色**：倖存者指揮官，率領英雄小隊對抗感染
- **基地**：倖存者避難所，玩家的作戰指揮中心

### 派系

| 派系 | 說明 |
|------|------|
| 倖存者聯盟 | 玩家所屬，由各地英雄組成 |
| 感染軍團 | 喪屍集群，由高階感染體指揮 |
| 未知勢力 | 第三方神秘組織（後期劇情） |

---

## 二、主線章節（規劃）

| 章節 | 名稱 | 主題 | 對應關卡 |
|------|------|------|---------|
| Ch.1 | 覺醒 | 基地遭襲，初次戰鬥 | 1-1 ~ 1-10 |
| Ch.2 | 突圍 | 外出尋找物資 | 2-1 ~ 2-10 |
| Ch.3 | 真相 | 發現病毒源頭線索 | 3-1 ~ 3-10 |
| Ch.4 | 背叛 | 內部叛徒揭露 | 4-1 ~ 4-10 |
| Ch.5 | 決戰 | 進攻感染核心 | 5-1 ~ 5-10 |

### 章節解鎖條件

每章最終關卡為 Boss 戰，通過後解鎖下一章。

---

## 三、角色背景故事（規劃）

每位英雄有：
- **簡介**：50 字內的角色定位描述
- **背景故事**：300 字觸發於首次獲得
- **好感度劇情**：5 段，隨好感度等級解鎖（未來功能）

---

## 四、劇情系統 UI（規劃）

| 元素 | 說明 |
|------|------|
| 對話框 | 底部彈出式，顯示角色立繪 + 台詞 |
| 選項分支 | 特定節點提供 2~3 個選擇 |
| CG 插畫 | 關鍵劇情全屏 CG |
| 劇情回放 | 設定面板可重看已解鎖劇情 |

### 劇情資料格式（規劃）

```typescript
interface DialogueLine {
  speaker: string        // 角色 ID 或 "narrator"
  text: string           // 台詞內容
  emotion?: string       // 表情變化
  choices?: Choice[]     // 分支選項
}

interface Choice {
  text: string
  nextNodeId: string
}

interface StoryNode {
  nodeId: string
  lines: DialogueLine[]
  nextNodeId?: string
  trigger?: 'stage-clear' | 'first-obtain' | 'affinity'
}
```
